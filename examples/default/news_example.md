## 行业动态

### OpenAI 发布 GPT-5 Turbo
全新的 GPT-5 Turbo 在推理速度和准确率上均大幅提升，支持 128K 上下文窗口，并在多项基准测试中刷新纪录。该模型现已在 API 端开放，开发者可直接调用。

来源：OpenAI Blog

### 国内大模型价格战白热化
继 DeepSeek 之后，百度文心一言、阿里通义千问相继宣布大幅降价，部分模型 API 调用成本降至原来的十分之一，进一步推动 AI 应用普及。

来源：36氪

---

## AI 工具与智能体更新

### Cursor 4.0 正式发布
新增多文件协同编辑、智能重构建议和内置终端 Agent 模式，编程效率显著提升。

来源：Cursor 官网

### Coze 发布工作流 2.0
新版工作流引擎支持条件分支、循环和子工作流嵌套，可构建更复杂的自动化 Agent 流程。

来源：字节跳动

| 功能 | 旧版 | 新版 |
|------|------|------|
| 节点类型 | 12种 | 20+种 |
| 最大嵌套 | 不支持 | 5层 |
| 调试方式 | 日志 | 可视化断点 |

## 模型发布与更新

### Claude 4.5 Opus 上线
Anthropic 发布 Claude 4.5 Opus，在代码生成、数学推理和多语言理解方面表现突出，尤其擅长长文档分析任务。

来源：Anthropic Blog

### Llama 4 开源发布
Meta 发布 Llama 4 系列开源模型，包含 8B、70B 和 405B 三个参数规模，在多项测试中接近商用模型水平。

来源：Meta AI Blog

---

## 重要研究进展

### Transformer++ 架构提出
微软研究院提出 Transformer++ 架构，通过改进注意力机制实现线性复杂度推理，在长序列任务上比原始 Transformer 快 3 倍。

来源：arXiv

### AI 生成代码可解释性突破
Google DeepMind 发布 Code Interpreter 报告，首次实现对 AI 生成代码的自动化解释，准确率达 87%。

来源：Google AI Blog
